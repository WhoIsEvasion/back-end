Урок 2: Переменные и типы данных

Теория:

В Python переменные используются для хранения данных.

Примеры типов данных:

  • Число = 10   #int(целое число)
  • Дробь = 3.14   #float(дробное число)
  • Текст = "Python"   #str(строка)
  • Логическое значение = True   #bool(логическое значение)
